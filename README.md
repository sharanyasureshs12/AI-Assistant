# 👁️ AI Assistant for Visually Impaired People

## 📖 About the Project

AI Assistant for Visually Impaired People is a computer vision-based application developed to help visually impaired individuals navigate more safely. The system detects objects in front of the user in real time and provides assistance to help avoid obstacles, enabling users to move around with greater confidence and independence.

## 🚀 Features

- Real-time object detection
- Detects obstacles in front of the user
- Assists visually impaired people in safe navigation
- User-friendly interface
- Camera-based environment monitoring

## 🛠️ Technologies Used

- Python
- HTML
- CSS
- JavaScript

## ▶️ Getting Started

1. Clone this repository.
2. Install the required dependencies.
3. Run the application:

```bash
python app.py
```

4. Open the application in your web browser.
